💧 JalDrishti — Water Security & Drought Intelligence Dashboard
📌 Overview

JalDrishti is an advanced, information-centric web dashboard designed to monitor and analyze water security and drought conditions across India. It integrates environmental data such as rainfall, temperature, reservoir levels, and groundwater status to provide real-time insights, risk assessments, and early warning alerts. The platform aims to support better decision-making for sustainable water resource management.

🚀 Features
🌍 Nationwide Monitoring
Covers multiple states with drought severity analysis
📊 Live Metrics Dashboard
Displays rainfall, temperature, reservoir levels, and groundwater data
🗺️ Interactive India Map
Visualizes drought severity using heatmaps and choropleth layers
⚠️ Alert System
Real-time alerts for extreme, severe, and moderate drought conditions
📈 Analytics & Insights
Monthly rainfall trends, anomaly detection, and top affected states
🔔 Notification Panel
Displays system alerts and updates
🌙 Dark/Light Mode
User-friendly theme toggle for better accessibility
🔍 State Search & Filtering
Quickly navigate and analyze specific regions
🧠 Tech Stack
Frontend: HTML, CSS, JavaScript
Visualization: Leaflet.js, Charts (custom/JS-based)
APIs: OpenWeatherMap (for live weather data)
Styling: Modern CSS (Glassmorphism + Responsive Design)
📂 Project Structure
JalDrishti/
│── index.html     # Main dashboard structure
│── style.css      # Styling and themes

⚙️ Setup Instructions

Clone the repository:

git clone https://github.com/your-username/jaldrishti.git

Navigate to the project folder:

cd jaldrishti

Add your API key in script.js:

OWM_API_KEY: 'YOUR_OPENWEATHERMAP_API_KEY'
Open index.html in your browser.
